Which are :

  1. Type
  2. Universal
  3. Id
  4. Class
Type
When we used type selector, we don't need any symbols for it. Just put the tag alone and it is enough to complete the 'type' selector.

As an example, you can see how i implement type selector. Right now, im used width attribute and apply it to table tag by using type selector. 
                                    table {
                                        width: 50%;
                                    }
Universal
We use Universal selector when we wanted to apply the CSS to the whole html code. How we do it? It is very easy. We just need to use asterisk symbol (*). Nothing more than that.
In this scenario, I used universal code to apply the thing as below :
  • Set font-family to Arial, Helvetica, sans-serif.
  • Set the font size to medium. 
                                * {
                                    font-family: Arial, Helvetica, sans-serif;
                                    font-size: medium;
                                }
Id
Id selector is used when we wanted to implement the CSS to the specific Id attribute. There are 2 things that we need to concern about.
  • Set Id attribute in the selected tag
    • This is example for Id attribute. 
I used id attribute and set it with "Id Example".
like this : < h4 id="IdExample">
  • Used hash(#) symbol to the id.
    • In the style tag(internal style), we will apply as below: 
                                    #IdExample {
                                    }
                                    Example :
                                    #IdExample {
                                    color: blue;
                                    }
    Class
    Class selector used when we wanted to apply the css into class attribute. 2 things that we need to concerns are:
  • Class value
  • Class symbol : period(.)
    • We will used period (.) to call the class attribute. Example : 
                                        class="ClassExample"
    This is how we call the call: 
                                        .ClassExample {}
    Now, I will apply CSS to this row using Class Selector as below: 
                                        background-color: cadetblue;
                                    font-family: Verdana, Geneva, Tahoma, sans-serif;
                                    font-weight: lighter;
                                    color: red;
